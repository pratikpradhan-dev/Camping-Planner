# 🏕️ Camping Trip Planner

A modern web application designed to help users plan and organize camping trips efficiently. From selecting destinations to managing trip details, this project simplifies the entire planning experience.

---

## 🚀 Features

* 📍 Browse and select camping destinations
* 🗓️ Plan trips with customizable details
* 🧭 User-friendly interface for smooth navigation
* 📱 Responsive design (works on mobile & desktop)
* ⚡ Fast and interactive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Tailwind CSS
* **Build Tools:** PostCSS
* **Other Tools:** VS Code, Git

---

## 📂 Project Structure

```
CampingTripPlanner/
```

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/pratikpradhan-dev/Camping-Planner.git
```

2. Navigate into the project:

```
cd CampingTripPlanner
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm run dev
```
